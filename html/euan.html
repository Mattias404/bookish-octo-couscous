Accessibility Statement

We have decided to commit our development of this website to be accessible to as wide a range of users as possible, regardless of technological experience and abilities.

This means that we have a wide variety of elements in mind whilst creating this website. For example, we have tried to select a colour pallette for our website that avoids colours that some users may be blind to, such as reds, greens and yellows.

We have also tried to use images that don’t have important details in such colours.

We use alternative text for images so users using a screen reader won’t miss out on content that non blind users can see.

For the navigation bar you will see that the text contrasts the background and that it is sizeable so difficulty should not arise when navigating throughout the web pages. We have used W3C guidelines [https://www.w3.org/TR/WCAG20/] as a reference for our accessible web design.
